Weather Predictions

Chart # 2

This weather bubble plot presents five years of daily data from the openweathermap API for the LWX (Baltimore/DC) office.
To create this graph we utilized the Vega library which uses JSON to transfer the commands to JavaScript.
There are four views: hourly in Fahrenheit or Celsius, and daily (F or C) with temperature on y, and time on x.
The x-ticks are weather phenomena (sun, snow, rain, fog, drizzle) for color, and precipitation as the size of the circle.
Below the bubble plot is a horizontal bar chart describing the weather phenomena (sun, snow, rain, fog, drizzle) & number of records of that phenomena.
